The Three Strategies
OilWar Active rebalancing

Each day the strategy computes a target allocation (0–100%) using drawdown depth and oil momentum signals, then immediately rebalances the portfolio to that target. It can go from 0% to 100% invested overnight, and can reduce exposure just as quickly. Best interpreted as: "how much of the portfolio should be in equities right now?"

OilWar Buy-Only accumulating

Uses the same signal logic but only ever buys — it will never reduce its equity position. When a buy signal fires, it deploys an incremental slice of the initial capital to purchase shares at the current price. The position grows over the window. The equity chart shows the mark-to-market value of shares held — what you would receive if you sold everything today.

Buy & Hold benchmark

Invests 100% of capital on the first day of the window and holds through to the end. No signals, no rebalancing. Used as the baseline to measure whether active strategies add value.

Signal Logic & Allocation Rules

The target allocation is built in three steps. First a base allocation is set from equity drawdown, then an oil modifier scales it, then a momentum boost may add to it.

Step 1 — Base Allocation (drawdown from 20-day high)
< 3% drawdown0% — market is near highs, no signal
3–5%25% allocation — mild pullback
5–7%50% allocation — moderate dip
7–10%75% allocation — significant drawdown
> 10%100% allocation — deep drawdown, full deployment
Step 2 — Oil Spike Modifier (vs 5-day average)
oil < −5%×1.10 — falling oil signals de-escalation, slightly boost allocation
−5% to +5%×1.00 — neutral, no adjustment
+5% to +20%×1.20 — moderate war-premium spike, add to position
> +20%×0.70 — extreme spike, reduce (tail-risk dampener)
Step 3 — Momentum Boost

If the equity 3-day return is >+1.5% and oil is not spiking hard (<10%), allocation gets a flat +25% bonus (capped at 100%). This rewards buying into a recovering market before the drawdown fully heals.

Charts Explained
OilWar Active — Daily Allocation %

Bar chart. Each bar shows what fraction of the portfolio (0–100%) was targeted on that trading day. Tall bars = fully deployed. Gaps = in cash. The rapid day-to-day variation reflects the strategy rebalancing to the current signal.

Buy-Only — Cumulative Cash Deployed %

Line chart. Shows the running total of capital committed as a percentage of the initial $100,000. It can only go up (or stay flat when no buy signal fires). When it reaches 100% the strategy has no more cash to deploy.

Equity Curves — All Strategies

All three strategies plotted on the same axes, rebased to $100,000 at the start of the window. This makes the return paths directly comparable regardless of any differences in initial capital timing. The Buy & Hold line is dashed grey; it is the benchmark you are trying to beat.

Oil & Stock Price — Same Period

Both raw price series are rebased to 100 at the first day of the window. The equity ticker (e.g. VTI) uses the left axis; oil (WTI crude) uses the right axis. This chart sits directly below the equity curves so you can visually correlate strategy decisions with the underlying price drivers. Dashed cyan line = oil; solid coloured line = equity.

Overview Table (◈ Overview tab)

One row per conflict. Columns show RTN (total return), MDD (max drawdown), and Sharpe for each strategy side by side. Use this to spot which wars each strategy handled well or poorly, and to compare strategies within the same window.

Performance Metrics
Return RTN

Total return from window start to end: (final − initial) / initial. Equity curves are rebased so this is always relative to the same $100,000 baseline.

Max Drawdown MDD

The largest peak-to-trough decline in portfolio value during the window. A max drawdown of −15% means the portfolio fell 15% from its highest point before recovering. Lower (less negative) is better.

Sharpe Ratio

Total return divided by annualised volatility. A rough measure of return per unit of risk. Above 1.0 is considered good; above 2.0 is excellent. Negative Sharpe means the strategy lost money. Note: calculated over a short window so values are noisy.

Final Value

The absolute dollar value of the portfolio at the end of the window, starting from $100,000.

Unrealized Position Box

This green-tinted box appears only for the OilWar Buy-Only strategy. Because it never sells, its performance cannot be realised as cash — it exists as a pile of shares.

Final Unrealized Value

Shares held × last closing price. What you would receive (before tax/fees) if you sold everything on the last day of the window.

Unrealized Return

The percentage gain or loss on cash actually deployed. If $60,000 was deployed and the position is now worth $72,000, the unrealised return is +20%.

Max Value Drawdown

The worst mark-to-market decline in the position's value during the window — not in cash terms, but as a fraction of peak unrealised value.

Caveats & Limitations
No transaction costs

The backtest does not deduct commissions, bid-ask spread, or market impact. Real-world returns would be lower, especially for OilWar Active which rebalances daily.

Look-ahead free, but short windows

Each window is isolated — signals only use data up to the current bar. However, windows of 6–12 months produce noisy Sharpe ratios and limited statistical significance.

Oil futures (CL=F) continuity

WTI front-month futures prices may show roll gaps. The 5-day average smooths most of this, but extreme spikes around contract rolls can generate spurious signals.
